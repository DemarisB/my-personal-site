Hi, I'm a developer that loves clean & elegant code.

A little bit about me.

I'm based in Cincinnati, Ohio where I was born and raised. My journey into tech started when I had a difficult time finding a full time job after graduating from college. I found that many jobs placed a higher value on job candidates with coding experience, even if the particular job was not tech based. So I began to research how to learn basic coding skills, mainly HTML & CSS. Once I found the right resources, I realized I wanted to build on those skills because I enjoyed the freedom and opportunity I saw in coding.

After working on different portfolio projects, I decided to build my first website. I was amazed at how with HTML, CSS and JavaScript I was able to build working sites! These initial steps into coding opened my eyes to the world of possibilities that a career in tech offers. I realized I had found something that offered the versatility and creativity I was looking for in my work. I have now made the switch to front-end development, where I can use my organizational skills and eye for detail to write clean and elegant code.

Hand Coded HTML

My focus is writing clean, well-formatted, semantic HTML5 by hand to make sure that the content is easy to read, easy to collaborate on, and trouble-shoot.

Well-Organized CSS

I pride myself on writing CSS that is well organized and easy to build on. I focus on keeping my CSS clean and fast to load, and I make it a habit to stay up to date on current best practices.

Ease Converting Designs into Code

You can trust me to take a designer's PSD and quickly and accurately convert it into a webpage that is a pixel-perfect match.
